First program - hello_world.js

1. Create file called hello_world.js with the following code.


		var http = require('http');
		http.createServer(function (req, res) {
		  res.writeHead(200, {'Content-Type': 'text/plain'});
		  res.end('Hello World\n');
		}).listen(1337, '127.0.0.1');
		console.log('Server running at http://127.0.0.1:1337/');

2. node hello_world.js

3. Browse to http://127.0.0.1:1337 in chrome.

4. Bask in the glory of how much easier than that was than Ruby and Rails has ever been.
